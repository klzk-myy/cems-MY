## Research Summary

**Found!** [obra/superpowers](https://github.com/obra/superpowers) is an agentic skills framework (180k stars) providing TDD, debugging, brainstorming, planning skills.

## Installation Method for Kilo Code

Kilo Code uses the `kilo plugin` CLI command for plugin installation:

```bash
# Install superpowers plugin
kilo plugin superpowers@git+https://github.com/obra/superpowers.git
```

This command will:
1. Resolve the plugin from git
2. Update your config file (kilo.jsonc or opencode.json)
3. Register all skills

2. **Run install command**: `kilo plugin superpowers@git+https://github.com/obra/superpowers.git`
3. **Restart Kilo** to load the plugin
4. **Verify**: Ask "Tell me about your superpowers"

## Alternative: Manual Config

If CLI fails, add to `~/.config/kilo/kilo.jsonc`:

```json
{
  "$schema": "https://app.kilo.ai/config.json",
  "plugin": ["superpowers@git+https://github.com/obra/superpowers.git"]
}
```

## Troubleshooting

If not working after install:
- Check logs: `kilo --print-logs --log-level DEBUG`
- Verify plugin line in config
- Use `kilo plugin` command to confirm installation
